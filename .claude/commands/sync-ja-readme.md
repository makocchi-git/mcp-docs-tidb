# Sync Japanese README

Reflect changes made to `README.md` into `docs/README.ja.md` in Japanese.

## Steps

1. Run `git diff README.md` to identify what changed.
2. Locate the corresponding section(s) in `docs/README.ja.md`.
3. Translate added or modified content into Japanese and apply it.
4. Report the changes made to the user.

## Rules

- Keep technical terms, env var names, and command names in English.
- Match the existing Japanese style (ですます調, 「。」「、」 punctuation).
- Do not translate code blocks.
- Preserve table structure.
